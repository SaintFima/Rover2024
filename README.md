# Rover2024

Arudino-based rover developed within 24 hours during the competition Hack-A-Bot 2024 at the University of Manchester

code for remote control using the HC-05 Bluetooth module.  Control of 4 motors, with binned commands for terminal or gamepad control. Plus servo control for the camera rotation in the x-axis for 180 degrees.

plus code for the ESP32 Cam. Creates own webpage and broadcasts live video from your cam with some additional features such as resolution, quality, AI.
